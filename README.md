## 🚀 快速开始

### 适合人群

- Windows系统用户想要学习ROS2
- 需要容器化ROS2开发环境的开发者
- 对Docker + ROS2组合感兴趣的技术人员
- 刚接触ROS2的新手开发者

### 前置要求

- Windows 10/11 (版本2004或更高)
- 已安装Docker Desktop
- 已安装WSL2
- 管理员权限

### 一分钟快速验证

```bash
# 启动一个临时ROS2容器
docker run -it --rm ros:humble bash

# 在容器内执行
source /opt/ros/humble/setup.bash
ros2
```
